== Description ==

Orange Front Page will make your regular theme in to a one page WordPress theme.

"Orange Front Page" is a set of front page block, like counters, team members and other.

Custom front page for "Orange Themes" created themes and also other themes offers bundle of customizable, lightweight and responsive building blocks that can be used to create exciting business portfolio websites.

Check out <a href="https://www.orange-themes.net/demo/orange-front-page/" rel="friend">DEMO</a>.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/orange-front-page` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. If you don't use this plugin with one of our WordPress.org themes then add this line <code><?php do_action( "orange_themes_front_page" ); ?></code> in your theme front-page.php or any template file that is used as front page.
5. Then go to the Appearance -> Customize section and there you will see the new custom front page options ( be sure that you have set a front page in Settings -> Reading -> Front page.
6. Enjoy it ;)
