== Description ==

"Orange Themes Custom Widgets" is a widgets set for One Page WordPress themes by <a href="http://www.orange-themes.com/" target="_blank" rel="friend">Orange-Themes.com</a>.

Widgets for "Orange Themes" is a bundle of customizable, lightweight and responsive building blocks that can be used to create exciting business portfolio websites. The widgets can be use with either the SiteOrigin Page Builder or any other widgetized area.

Check out <a href="http://www.orange-themes.net/demo/asterion/" rel="friend">DEMO</a>.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/orange-themes-widgets` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. Then go to the Appearance -> Widgets section and there you will see the new custom widgets.
5. Enjoy it ;)
